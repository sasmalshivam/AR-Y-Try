📋 HOW TO USE - COPY & PASTE EXAMPLES:

A) PRODUCT SHOWCASE (Priority #1 - Most Important)
   Use: AnimatedProductsGallery component
   
   Code:
   ----
   import AnimatedProductsGallery from './components/AnimatedProductsGallery';
   
   export default function ProductList() {
     return <AnimatedProductsGallery />;
   }
   ----

B) HERO SECTION (Priority #2)
   Use: StaggerContainer + FloatingElement + ScrollIndicator
   
   Code:
   ----
   import { StaggerContainer, FloatingElement, ScrollIndicator } 
     from './components/AnimationComponents';
   
   export default function HomeHero() {
     return (
       <>
         <StaggerContainer delay={0.2}>
           <h1>Welcome to AR-Y-TRY</h1>
           <p>Try Before You Buy</p>
         </StaggerContainer>
         
         <FloatingElement duration={3}>
           <div className="floating-orb"></div>
         </FloatingElement>
         
         <ScrollIndicator />
       </>
     );
   }
   ----

C) STORY SECTIONS (Priority #3)
   Use: RevealFromLeft/Right + StatCard
   
   Code:
   ----
   import { RevealFromLeft, RevealFromRight, StatCard } 
     from './components/ScrollAnimations';
   
   export default function HomeStories() {
     return (
       <>
         <RevealFromLeft delay={0.1}>
           <h2>Our Mission</h2>
           <p>Description...</p>
         </RevealFromLeft>
         
         <RevealFromRight delay={0.2}>
           <img src={storyImage} />
         </RevealFromRight>
         
         <StatCard icon="👥" value={50000} label="Users" delay={0.3} />
       </>
     );
   }
   ----

D) COMPLETE DEMO (See all animations working)
